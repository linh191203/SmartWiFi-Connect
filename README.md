# SmartWiFi-Connect

Ứng dụng Android hỗ trợ kết nối Wi-Fi thông minh bằng:
- Quét mã QR Wi-Fi
- Quét ảnh chứa tên Wi-Fi và mật khẩu bằng OCR
- Nhập tay thông tin Wi-Fi
- Xác nhận kết quả trước khi kết nối
- Lưu lịch sử

## Công nghệ
- Kotlin
- Jetpack Compose
- CameraX
- ML Kit
- Room
- Navigation Compose
